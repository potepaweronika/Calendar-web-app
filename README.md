# Calendar-web-app

### Opis funkcjonalności: 
* tworzenie konta użytkownika
* aktualizacja konta 
* usunięcie konta 
* tworzenie wydarzeń w kalendarzu
* aktualizacja wydarzenia
* usunięcie wydarzenia

### Wykorzystane technologie:
* frontend
  * React
* backend
  * JavaScript
* baza danych: 
  * Mongo
